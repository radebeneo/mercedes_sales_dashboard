# Mercedes-Benz Global Sales Analysis (2020-2025)

An interactive data visualization dashboard designed to analyze global sales performance and market trends for Mercedes-Benz. Using a comprehensive dataset of ~1.8 million sales records, this project identifies key drivers in the luxury automotive sector, tracking the transition from internal combustion engines to electrification.

### Core Objectives:
* **Trend Analysis:** Monitoring sales volume fluctuations from 2020 through 2025.
* **Product Performance:** Identifying top-selling models (GLC, GLE, S-Class) and luxury segment resilience.
* **Electrification Tracking:** Visualizing the shift toward Hybrid and Electric (EV) powertrains.
* **Pricing Dynamics:** Analyzing base price evolution and the impact of inflation on luxury car segments.
